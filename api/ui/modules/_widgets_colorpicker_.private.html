hsvToRgb

  • hsvToRgb(hue: number, saturation: number, value: number): [number, number, number]
    • Defined in widgets/colorpicker.ts:368

    Convert a given color in HSV space to RGB space

    We need this, since the ImageData API only works in RGB space (whereas the color wheel must be in HSV space).

    see

    https://en.wikipedia.org/wiki/HSL_and_HSV#HSV_to_RGB

    Parameters

    • hue: number

      The hue, in degrees, of the color

    • saturation: number

      Saturation, in the range [0-1]

    • value: number

      Value, in the range [0-1]

    Returns [number, number, number]

    A 3-tuple of [r, g, b], each element being in the range [0-255]
