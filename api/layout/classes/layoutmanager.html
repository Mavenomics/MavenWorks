Class LayoutManager

Base layout for Maven Dashboards

This class manages the layout operations of a set of widgets, which are arranged in a tree and consist of DashboardLayoutRegions. These regions have a set of layout properties that describe appearance and layout behavior- for instance, all regions have properties like borderWidth.px and borderColor.
